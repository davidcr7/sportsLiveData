# 直播吧赛程比分 Chrome 插件

将 [直播吧 zhibo8.com](https://www.zhibo8.com/) 首页的比赛直播数据嵌入浏览器弹窗：

- **赛程列表**：解析直播吧首页赛程（今天及未来几天），显示时间、赛事、对阵、球队 logo
- **实时比分**：弹窗打开期间每 10 秒自动刷新比分与比赛状态（进行中 / 完赛 / 延期）
- **悬浮详情**：鼠标悬浮到某场比赛上，显示实时比分、比赛时段、半场比分，以及**进球球员和进球时间**（足球），未开赛比赛显示赛事标签与直播频道
- **Tab 切换**：全部 / 足球 / 篮球 / 电竞 / 综合 / 完赛
- **日期切换**：类型 Tab 下方有日期胶囊（全部日期 / 今天 / 明天 / 后天…，随数据动态生成），与类型筛选叠加生效
- **固定 Tab**：悬浮 Tab 后点击 📌 即可固定，之后每次打开插件默认显示固定的 Tab（再点一次取消固定）
- 点击比赛行可直接打开直播吧对应的直播页

## 安装方法

1. 打开 Chrome，地址栏输入 `chrome://extensions/`
2. 打开右上角「开发者模式」开关
3. 点击「加载已解压的扩展程序」，选择本目录（`sportsLive`）
4. 工具栏出现红色足球图标，点击即可使用

## 数据来源

| 数据 | 接口 |
|---|---|
| 赛程列表 | `https://www.zhibo8.com/` 首页 HTML（`.schedule` 内嵌赛程） |
| 实时比分（全部比赛） | `https://bifen4pc2.qiumibao.com/json/list.htm` |
| 单场详情（进球人 / 进球时间） | `https://bifen4pc.qiumibao.com/json/{日期}/{比赛id}.htm` |

以上均为直播吧页面自身使用的公开接口，插件通过 `host_permissions` 直接跨域请求，无需后台服务。

## 目录结构

```
manifest.json    MV3 清单
popup.html/css/js  弹窗界面与逻辑
icons/           插件图标
test/harness.html  本地测试挂架（stub 掉 chrome API 与网络，用 fixtures 数据调试界面）
```

调试界面样式时可运行 `python3 -m http.server 8734 --directory .` 后打开
`http://localhost:8734/test/harness.html`，无需反复重载扩展。

## 说明

- 比分接口约 10 秒更新一次，与直播吧官网一致
- 「完赛」Tab 依赖实时比分接口返回的状态，只包含比分接口中出现过的比赛
- 若直播吧调整首页结构或接口地址，需要相应更新 `popup.js` 顶部的常量与解析逻辑
