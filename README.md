# 直播吧赛事悬浮窗浏览器插件

点击浏览器工具栏图标，会弹出一个悬浮面板，展示直播吧各分类里的赛事信息。

## 功能

- 自动读取直播吧最新赛事
- 支持全部、重要、完赛、足球、篮球、电竞、综合分类
- 可固定任意分类，下次打开自动切换到该分类
- 每 30 秒自动刷新实时比分和比赛阶段
- 按日期切换赛程
- 展示开赛时间、赛事名称、球队、比分和队徽
- 鼠标悬浮或点击比分，查看进球、点球、乌龙、红牌、两黄变红和失点事件
- 保留直播、文字、比分、动画等原站入口
- 请求失败时自动使用最近一次成功缓存

## 固定默认分类

先切换到想固定的分类，再点击分类栏最右侧的图钉。图钉变蓝后，该分类会成为默认分类；以后每次打开插件都会直接展示它。再次点击图钉可以取消固定。

## 安装

1. 在 Chrome 地址栏打开 `chrome://extensions/`
2. 打开右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择本项目文件夹
5. 把“直播吧赛事悬浮窗”固定到工具栏

插件读取直播吧首页和直播吧使用的实时比分接口，并在浏览器本地保存最近一次成功结果作为断网兜底。
